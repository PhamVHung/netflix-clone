# MoonFlix

    Bài tập lớn môn Lập trình Web

# How to run
1. Install dependencies
   - For client
   ```bash
   cd client/
   npm install
   ```
   - For server
    ```bash
   cd server/
   npm install
   ```
2. Configure `.env` file in the `server` folder
    ```
    MONGODB_URL=
    PORT=
    TOKEN_SECRET=
    TMDB_BASE_URL=
    TMDB_KEY=
    ```
3. Run program
    ```bash
    cd client
    npm start
    ```
